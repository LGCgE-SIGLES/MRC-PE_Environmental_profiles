# Overview of data sources used in the MRC-PE project

This table provides a synthetic overview of all environmental data sources mobilized in the MRC-PE project.\
For each thematic domain, it lists the corresponding sub-thematic components and the original data sources.

| THEMATIC DOMAIN                                 | SUB-THEMATIC                                       | DATA SOURCE                                                                                                  |
|-------------------------------------------------|----------------------------------------------------|--------------------------------------------------------------------------------------------------------------|
| **Contamination levels of environmental media** | Air pollution                                      | *Institut national de l'environnement industriel et des risques (INERIS)*                                    |
|                                                 | Soil contamination                                 | *GIS SOL – Réseau de Mesures de la Qualité des Sols (RMQS)*                                                  |
| ---                                             | ---                                                | ---                                                                                                          |
| **Proximity to emission and nuisance sources**  | Contaminated sites and soils                       | *Carte des Anciens Sites Industriels et Activités de Services (CASIAS)*                                      |
|                                                 | Facilities classified for environmental protection | *Système d'information de l'inspection des installations classées (S3IC)*                                    |
|                                                 | Industrial accidents                               | *Analyse, Recherche et Information sur les Accidents (ARIA)*                                                 |
|                                                 | Road network                                       | *Institut national de l'information géographique et forestière (IGN) – BD TOPO*                              |
|                                                 | Waste treatment facilities                         | *Agence de l'environnement et de la maîtrise de l'énergie (ADEME) – SINOE*                                   |
|                                                 | Wastewater treatment plants                        | *Agence de l'environnement et de la maîtrise de l'énergie (ADEME) – Portail sur l'assainissement collectif*  |
| ---                                             | ---                                                | ---                                                                                                          |
| **Land use**                                    | Agricultural land use                              | *Institut national de l'information géographique et forestière (IGN) – Registre Parcellaire Graphique (RPG)* |
|                                                 | Biophysical land use                               | *Agence européenne pour l'environnement (AEE) – Corine Land Cover (CLC)*                                     |
|                                                 | Land take dynamics                                 | *Centre d'études et d'expertise sur les risques, l'environnement, la mobilité et l'aménagement (CEREMA)*     |
| ---                                             | ---                                                | ---                                                                                                          |
| **Agricultural practices**                      | Crop treatment frequency index                     | *Solagro – Indice de Fréquence de Traitement (IFT)*                                                          |
|                                                 | High nature value agricultural systems             | *Solagro – Haute Valeur Naturelle (HVN)*                                                                     |
|                                                 | Purchases of pesticide active substances           | *Banque Nationale des Ventes de produits phytopharmaceutiques par les Distributeurs agréés (BNV-D)*          |
|                                                 | Stocking density                                   | *Agreste – Recensement général agricole (RGA)*                                                               |
| ---                                             | ---                                                | ---                                                                                                          |
| **Environmental amenities**                     | Access to green spaces                             | *OpenStreetMap (OSM)*                                                                                        |
|                                                 | Naturalness index                                  | *Union internationale pour la conservation de la nature (UICN) – CartNat*                                    |
|                                                 | Normalized Difference Vegetation Index (NDVI)      | *Santé publique France (SPF) – data available upon reasonable request (Adélaïde et al., 2024)*               |
| ---                                             | ---                                                | ---                                                                                                          |
| **Climate**                                     | Temperature                                        | *Santé publique France (SPF) – data available upon reasonable request (Adélaïde et al., 2024)*               |
